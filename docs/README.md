## Goal

Make it easy for anyone to create and share short, replayable microgames without depending on the main Indie Cake platform.

## Platform defaults

- Target play area: `320x240`
- Preferred engine package: [indiecake-engine on npm](https://www.npmjs.com/package/indiecake-engine)
